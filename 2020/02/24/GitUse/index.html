Efficient Method | Git常用命令

2020-02-24 /
  • Git常用指令
 / 点击 / 阅读耗时 1 分钟

一、git 拉取项目步骤

1. 新建项目目录

mkdir [ProjectName]

2. git 初始化

git init

3. 绑定项目所在的地址

git remote add origin [url]

4. 拉取master

git pull origin [master]

5. 新建并关联远程分支

git checkout -b [本地分支名] [远程分支名-origin/**]

二、git 日常提交步骤

1. git add .

2. git commit -m ''

3. git pull origin [branchName]

4. git push origin [branchName]

5. git push -u origin [branchName]

三、其他常用指令

1. 在本地目录更新所有的远程分支

git fetch
