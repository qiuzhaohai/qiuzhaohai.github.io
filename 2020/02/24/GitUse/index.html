QiuZH's Efficient Method | Git常用命令

2020-02-24 /
  • Git常用指令
 / 点击 / 阅读耗时 2 分钟

一、git 拉取项目步骤

1. 新建项目目录

mkdir [ProjectName]

2. git 初始化

git init

3. 绑定项目所在的地址

git remote add origin [url]

4. 拉取master

git pull origin [master]

5. 新建并关联远程分支

git checkout -b [本地分支名] [远程分支名-origin/**]

二、git 日常提交步骤

1. 把代码放入仓储的门口(暂存区)

git add .

2. 把仓储门口的代码放到仓储房间中（版本库）

git commit -m ''

3.拉分支代码

git pull origin [branchName]

4. 推本地分支到远程分支代码

git push origin [branchName]

5. 推本地分支强制覆盖远程分支代码

git push -u origin [branchName]

三、其他常用指令

1. 在本地目录更新所有的远程分支

git fetch

2. 创建分支

git branch dev

3. 切换分支

git checkout dev

4. 查看分支

git checkout dev

5. 合并分支

git merge dev

6. 查看当前本地暂存区状态

git status

7. 查看日志

git log
